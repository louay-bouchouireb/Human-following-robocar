# Human-following-robocar
Human following robocar

This project aims to build a fully autonomous robocar, that follows humans, animals...

3 x HCSR04 Ultrasonic sensors.
1 x Esp32 .
23 x Wires.
2 x Dc motors.
1 x Breadboard.
1 x Battery holder.
1 x 3D printed frame.
3 x 3D printed UltraSonic sensors holders.
1 x Marble ball.
1 x 3D printed marble ball holder.
3 x lithium batteries
